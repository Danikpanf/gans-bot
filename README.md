# Solor Piece Bot

Telegram бот для магазина игровых товаров Solor Piece.

## Установка

```bash
cd bot
npm install
```

## Настройка

Создайте файл `.env` и укажите:

```
BOT_TOKEN=ваш_токен_бота
WEB_APP_URL=https://ваш-домен.com
ADMIN_CHAT_ID=ваш_id_администратора
```

## Запуск

```bash
npm start
```

Для разработки:

```bash
npm run dev
```
